Step 1: Go to Render

Visit:

https://render.com

Click Get Started.

Step 2: Login

Choose

Continue with GitHub

Authorize Render to access your GitHub account.

Step 3: Create a New Web Service

Click

New +

↓

Web Service
Step 4: Select Repository

Choose

AI-Mood-Tracker

Click

Connect
Step 5: Fill the Details
Name
ai-mood-tracker

or any name you like.

Region

Choose the nearest one.

Example:

Singapore
Branch
main
Runtime
Python 3
Root Directory

Leave empty.

Step 6: Build Command

Enter

pip install -r requirements.txt
Step 7: Start Command

Since your project uses Flask (app.py), enter:

gunicorn app:app

If app.py contains:

app = Flask(__name__)

then the above command is correct.

Step 8: Environment Variables (Optional)

If your app uses

os.environ.get(...)

add them here.

Otherwise, leave empty.

Step 9: Create Web Service

Click

Deploy Web Service

Render will now

Install packages
Build your project
Start the server

This usually takes 2–5 minutes.

Step 10: Open Your Website

You'll receive a URL like

https://ai-mood-tracker.onrender.com

Open it in your browser.

If Deployment Fails

Open the Logs tab.

The error will usually tell you what's missing.

Important: Add gunicorn

Open your requirements.txt.

Make sure it contains:

Flask
gunicorn

If gunicorn is missing, install it locally:

pip install gunicorn

Then update your requirements:

pip freeze > requirements.txt

Commit and push the changes:

git add .
git commit -m "Added gunicorn"
git push

Render will automatically redeploy.

Final Checklist
